## (EN) Business Model

### 1.1 Project Overview

A premium aesthetic, wellness, and longevity clinic based in Thailand, collaborating with renowned Korean dermatology, plastic surgery, and wellness centers. The core model is online consultation with Korean specialists → delivery of globally trained, minimally invasive treatments. Future roadmap includes personalized wellness/longevity programs including stem cell therapy.

|---|---|---|
| **Procedure income** | Dermal treatments, cosmetic procedures, wellness treatments | 55–65% |

| **Partnership revenue** | Korean clinic collaboration fees, patient referrals, joint programs | 10–15% |
| **Product sales** | Skincare lines, supplements, stem-cell-based products | 5–10% |

| Cost Item | Description | Est. Share (assumption) |
|---|---|---|

| **Rent** | Premium clinic space rental (Bangkok/Chiang Mai) | 10–12% |
| **Marketing** | Digital marketing, KOL collaboration, medical tourism channels | 8–12% |
| **Compliance** | Medical licensing, insurance, audits, legal costs | 5–8% |
| **Operations/admin** | Admin, IT systems, CRM | 5–7% |

### 4.1 배분 워터폴

| 순위 | 항목 | 설명 |
|---|---|---|
| 1 | 운영 비용 | 인건비, 임대료, 장비, 마케팅, 보험 등 |

| 3 | Senior 쿠폰 | CLINIC-SR 보유자에게 연 8–12% 기준 월별 배분 |
| 4 | Growth 배분 | CLINIC-GR 보유자에게 잔여 EBITDA의 30–45% 배분 (KPI 연동) |
| 5 | 잔여 이익 | 클리닉 재투자 또는 다음 기 이월 |

|---|---|---|

| 운전자금 버퍼 | 월 매출의 5% | 계절적 매출 변동, 긴급 운영 비용 |

|---|---|

## (KR) 의료 및 규제 리스크

### 5.1 인허가 (Licensing)

| 리스크 항목 | 설명 | 완화 방안 |
|---|---|---|
| 태국 의료 면허 | 외국인 의사 진료 제한, 태국 의사 면허 필요 | 태국 면허 보유 의사 상주 + 한국 의사는 자문·온라인 역할로 한정 |
| 클리닉 운영 허가 | 태국 보건부 허가 필요 (Private Hospital/Clinic License) | 개원 전 모든 인허가 취득 완료 후 자금 집행 |

### Payout Module
```solidity
// Pseudocode — not audited
contract ClinicPayoutModule {
    function executeMonthlyDistribution(
        uint256 _clinicRevenue,
        uint256 _operatingExpenses,
        uint256 _reserveAllocation
    ) external onlyAdmin {
        uint256 netIncome = _clinicRevenue - _operatingExpenses - _reserveAllocation;
        
        // Senior coupon (priority)
        uint256 seniorCoupon = calculateSeniorCoupon(); // annualized / 12
        uint256 seniorPayout = min(netIncome, seniorCoupon);
        
        // Growth distribution (residual, KPI-linked)
        uint256 residual = netIncome - seniorPayout;
        uint256 growthRate = getKPIStepUpRate(); // 30–45% based on KPI
        uint256 growthPayout = residual * growthRate / 100;
        
        // Distribute via claim contracts
        distributeToClaim(seniorClaimContract, seniorPayout);
        distributeToClaim(growthClaimContract, growthPayout);
        
        emit MonthlyDistribution(seniorPayout, growthPayout, block.timestamp);
    }
}
```
